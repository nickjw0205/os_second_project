소프트웨어 보증서
green-01

본 소프트웨어는 Apache License 2.0이 적용된 오픈소스 프로그램으로 소프트웨어 컴포넌트(component) 또한 오픈소스로 작성되었다.
여러 컴포넌트에 적용된 라이센스(license)는 MIT License, Apache License 2.0, BSD-3-License로 아래의 사항을 공통적으로 만족한다.

1.	복제, 배포, 수정의 권한 허용
2.	상업적 이용 가능
3.	소스코드 공개의 의무 없음
4.	저작권 표시
5.	기타 등등

해당 내용은 소프트웨어의 상업적 사용에 있어 문제를 일으킬 요소가 없다.
또한 공통 사항 외에 각 라이센스에 포함된 내용 또한 소프트웨어 간의 저작권 충돌 발생 부분이 없으므로, 라이센스 간 법적 문제 유발 요소는 없다.
(사용 오픈소스 프로그램과 라이센스는 본 소프트웨어 내의 LICENSE.txt 파일과 NOTICE.txt 파일에서 확인 가능하다.)
따라서 본 소프트웨어 사용자가 라이센스 내용을 준수하여 소프트웨어를 사용했을 경우 문제가 없음을 보증하며, 법적 문제가 발생할 경우 책임이 소프트웨어 제작자에 있음을 명시하는 바이다.
(단, 컴포넌트 소프트웨어 제작자인 저작권자에게는 책임이 없다.)
